⚙️ Setup guide
One-time setup (on your local machine):
  1. Clone Playwright MCP and Seedance skills (see SEEDANCE_SETUP.md in the backend repo)
  2. Register the Playwright MCP server in ~/.claude/settings.json
  3. Restart Claude Code and say: "Generate a Seedance cinematic ad for RileyAI"
  4. When Claude finishes, the MP4 downloads to your machine — drop it in the upload box below
